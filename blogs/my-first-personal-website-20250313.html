My first personal website

March 21, 2025 5 min read Technology, Hobbies 0 views

If you know me personally, you know that I'm very into programming and stuff. I've been self-learning all sorts of programming languages and frameworks for a while now, and I've always wanted to have a personal website to showcase my projects and share my thoughts with the world.

Building this website was a fun and challenging experience. I wanted to create a clean and modern design that reflects my personality and style. I also wanted to make sure that the website is easy to navigate and user-friendly.

One of the biggest challenges I faced was deciding on the color scheme and layout of the website. I spent hours browsing through different websites and design inspiration to find the perfect combination of colors and fonts. I also spent a lot of time tweaking the layout and structure of the website to make sure that it looks good on all devices.

Overall, I'm really happy with how the website turned out. I think it's a great representation of who I am and what I do. I'm excited to start sharing my projects and thoughts with you all, and I hope you enjoy exploring my website!

If you want to learn more about how I built this website, go to my Projects page about this website! I shared more details about how I built this website and the technologies I used.
