Personal Website

March 17, 2025
Web

1. Project Overview

This project is a personal website that I created to showcase my projects, blog posts, and other information about me. The website is built using HTML, CSS, and JavaScript, and is hosted on Cloudflare Pages (for now)

2. Key Features

  • Responsive site design that works on all devices
  • No trackers or ads, ensuring privacy
  • Counters for page views and unique visitors
  • Filters for projects and blog posts
  • Easy to use contact form

3. Technical Details

The website is built using HTML, CSS, and JavaScript. The site is hosted on Cloudflare Pages, which provides free hosting with a custom domain. The site uses no trackers or ads, ensuring privacy for visitors.

  • HTML: Used for the structure of the site
  • CSS: Used for styling and layout
  • JavaScript: Used for interactivity and dynamic content
  • Cloudflare Pages: Used for hosting the site and basic domain protection and analytics

4. Development Process

I came up with the idea for this website after realizing that this will make for a really good project to showcase my skills, a place where I can share my projects and blog posts, and an interesting project that I can put in my academic portfolio.

I started by consulting with AI, Claude 3.7 Sonnet Thinking in Github Copilot, to get some ideas on how to design the website, as well as some examples. I then started coding the website from scratch, using HTML, CSS, and JavaScript.

Github Copilot Screen Click the image to zoom in!

Since I already had a domain on Cloudflare, I decided to host the site with Cloudflare Pages. I plan to move the site to my own server in the future, but for now, Cloudflare Pages is a great free option with unbeatable uptime and convenience.

Cloudflare Pages Screen Click the image to zoom in!

My initial plan for the website is to showcase my personal projects and blogs. The template by Claude was more than enough for me, so I then focused on adapting the template for my own use. It then transformed into the page you are on now!
